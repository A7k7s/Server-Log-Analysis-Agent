# Agentic Server Log Analysis using Groq

This project is an **Agentic AI-based Server Log Analysis System**.  
Instead of running a fixed script, this system behaves like an **intelligent agent**:

- It understands what stage of analysis the system is in  
- Decides which operation to perform next  
- Processes large logs intelligently  
- Generates meaningful insights automatically  

The agent does not blindly follow a pipeline.  
It **thinks, decides, and acts** step-by-step like a real AI system.
